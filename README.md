# PoB Injector

A tool to seamlessly evaluate Path of Exile 2 items directly from your browser into Path of Building.

## 🌟 Features
- **Runes & Sockets:** Apply runes individually or fill missing sockets on the fly.
- **Amulet Anoints:** Override amulet anoints directly from the UI.
- **Smart Sorting:** Click any PoB stat (like "Average Damage") to sort all visible trade results automatically.
- **1-Click Import:** Send any item from the web directly into your PoB build.

## 🚀 Installation

### 1. Local Server (Required)
1. Download the latest `PoB_Injector_Release_vX.X.X.zip` from [Releases](https://github.com/rauldzmartin/PoB-Injector/releases/latest).
2. Extract the ZIP anywhere on your PC and run `PoB-Injector.exe`.
3. The server will silently run in your system tray and auto-detect your active PoB build.

### 2. Browser Extension (1-Click Install)
The easiest way to use PoB Injector without dealing with Chrome Developer Mode is via **Tampermonkey**.

1. Install the free **[Tampermonkey](https://www.tampermonkey.net/)** extension for your browser.
2. Go to the [Releases](https://github.com/rauldzmartin/PoB-Injector/releases/latest) page and click on `pob-injector.user.js`.
3. Tampermonkey will prompt you to install it. Click **Install**.
4. You're done! The script will even auto-update itself in the future.

*(If you prefer the manual way, you can still load the `extension` folder from the ZIP using Chrome's Developer Mode).*
